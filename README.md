# Dynamical-Bestiary-Cahn-Hilliard-Equation
Python notebooks for the 2-dimensional Cahn-Hilliard equation

$$
\partial_t c = M \nabla^2 \big(f'(c) - \epsilon^2 \nabla^2 c\big)
$$

on a periodic square domain.
